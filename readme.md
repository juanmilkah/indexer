# Search Engine

## Features
- Indexing pdf files in a directory  
- Searching of terms

## Building the binary(Linux-based-platforms)
```bash
bash build.sh
```

## Usage

Indexing 
```bash
indexer index documents/
```

Searching
```bash
indexer search index.json "foo bar baz"
```
